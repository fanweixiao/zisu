shareposter-WordPress文章海报生成插件

zisu 2019-04-06 49

shareposter是一款为wordpress文章自动生成分享海报的插件

shareposter是一款能自动给文章生成一张分辨率为750*1366海报的插件，海报样式精美且自带文章链接的二维码，有利于在其他平台的分享和传播。
海报生成主要由后端处理，为节省服务器资源shareposter提供Ajax接口来获取海报，在启用插件后Ajax接口自动生成，在前台你只需要在需要展示海报的位置发起Ajax请求即可快速获取当前文章的海报链接。

使用

接口地址：

你的网站域名/wp-admin/admin-ajax.php?action=gen_poster

提交参数：
向接口提交当前文章id

{id:id}

接口返回示例：

{
    "code":1,
    "title":"shareposter-Wordpress文章海报生成插件",
    "poster":"https://www.lylares.com/shareposter/fcf2f53ba9e4fb87bba7d2c4084596be.png"
}

说明

  1. 自动获取文章特色图片为海报顶部图片
  2. 海报统一保存在插件目录下的「poster」文件夹
  3. 同一篇文章根据发布日期保存一张海报，多次前台请求将返回同一张海报
