使用virt-what判断VPS虚拟化技术

zisu 2019-04-10 907

说明：VPS拥有多种虚拟化技术，常见的VPS虚拟化架构有OpenVZ、Xen、KVM这三种，我们可以用“virt-what”这个工具，判断和参考自己的VPS虚拟化架构。

1、下载工具

wget http://www.moerats.com/usr/down/virt-what-1.11.tar.gz

2、安装

tar zxf virt-what-1.11.tar.gz   #解压缩包
cd virt-what-1.11               #进入目录
./configure                     #按默认设置
make && make install            #编译并安装

3、查看

virt-what
